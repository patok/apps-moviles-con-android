La plataforma Android

  • Android es una plataforma (entiéndase pila de software) que permite correr aplicaciones en dispositivos móviles: teléfonos, tablets, relojes(wearables).
  • Opera con distintos componentes a distintos niveles: kernel SO, librerías de sistema, framework de aplicaciones, y aplicaciones principales (marcador teléfono, gestor de contactos, navegador, cámara de fotos).
  • También provee herramientas para el desarrollo de aplicaciones: SDK (soft development kit), que incluye librerías y herramientas de desarrollo.
  • Provee además un montón de documentación!

Servicios estándares

    típicamente se encargan de:procesos intensivos en uso de cpu y performance (rendering y actualización del display)

    • System libC: creación procesos e hilos, alocación de memoria, funciones/rutinas matemáticas

    Dalvik VM

    La app se ejecuta dentro de la DVM. Uno puede asumir que el código Java se compila, y se ejecuta en standard Java virtual machine, pero no es así.

    • escribe la aplicación en lenguaje Java
    • es compilado a un .class (bytecode standard)
    • una herramienta llamada DX transforma el bytecode Java, en un fichero ùnico con formato llamado DEX
    • fichero DEX file es empaquetado con los resources de la app e instalado en dispositivo
    • el usuario ejecuta la app, la DVM entonces ejecuta las clases del DEX file

        • Esto es así porque la DVM fué específicamente diseñada para ejecutarse en dispositivos de recursos reducidos: CPU mas lentas, menos memoria, batería limitada.

    Application Framework

    Software reusable que las aplicaciones móviles requieren.

    • Package Manager: base de aplicaciones instalada
    • Window Manager: se ocupa de gestionar ventanas de apps
    • View System: contiene componentes gráficos de la IU
    • Resource Manager: maneja recursos de las app - layouts, graficos, texto localizado
    • Activity Manager: se encarga de las Activities
    • Content Providers: base de datos que permite a las apps almacenar y compartir datos
    • Notification Manager: se encarga la barra de notificaciones (eventos desde las apps)
