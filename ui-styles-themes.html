Estilos y Temas (Styles & Themes)

Estilo (style): un estilo especifica las propiedades visuales de los widgets/elementos de la interface de usuario, como : color, altura, padding, tamaño fuente, etc.
Tema (theme): un tema constituye un mecanismo para aplicar de manera consisten un estilo a una applicación o a una activity. Cuando un estilo es aplicado como tema, cada view en la Activity o Application aplica cada propiedad del estilo que soporta

Style

Por ejemplo utilizando un style se puede llevar este layout xml: 


<TextView
    android:layout_width="fill_parent"
    android:layout_height="wrap_content"
    android:textColor="#00FF00"
    android:typeface="monospace"
    android:text="@string/hello" />
a este otro: 

<TextView
    style="@style/CodeFont"
    android:text="@string/hello" />

Style: definición

Se define en un recurso XML bajo res/values/, de nombre arbitrario, el nodo raíz debe ser <resources>. 


<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="CodeFont" parent="@android:style/TextAppearance.Medium">
        <item name="android:layout_width">fill_parent</item>
        <item name="android:layout_height">wrap_content</item>
        <item name="android:textColor">#00FF00</item>
        <item name="android:typeface">monospace</item>
    </style>
</resources>
agregar un <style> con attr name para identificarlo; debajo cada <item> que corresponde por attr name con la propiedad y en el texto el valor correspondiente. Este valor puede ser un keyword string, un color hexa, una referencia a otro recurso, dependiendo de la propiedad

Style: herencia

El atributo parent permite especificar un tema del cual heredar las propiedades. De esta manera uno obtiene las definiciones de las propiedades en el tema padre, y puede definir las específicas del tema que desea cambiar o agregar. 


    <style name="GreenText" parent="@android:style/TextAppearance">
        <item name="android:textColor">#00FF00</item>
    </style>

para temas definidos por el desarrollador se puede prescindir del attr parent: 


    <style name="CodeFont.Red">
        <item name="android:textColor">#FF0000</item>
    </style>

Style: aplicando estilos

Existen 2 maneras de aplicar estilos:

  • a una View individual: usando attr style
  • a una Activity o Application : attr android:theme

Style: aplicando estilos

View


<TextView
    style="@style/CodeFont"
    android:text="@string/hello" />
al aplicar a una View o ViewGroup individual, los views hijos NO hereden sus propiedades

Application


<application android:theme="@style/CustomTheme">
al aplicarse al Application o Activity se heredan las propiedades del tema en todas las views hijas de estas
