Menúes

Los menúes son elementos comunes de la UI que permiten al usuario invocar acciones en la app, para proveer una experiencia de usuario consistente Android provee el Menu API para presentar estas opciones en una Activity.

Existen 3 tipos básicos de menúes:

  • menú de opciones (Options Menu) y barra de acciones (action bar - API level 11 en adelante)
  • menú contextual y modo de acción contextual
  • menú de ventana emergente (popup menu)

Options menu and action bar

El menú de opciones es la colección principal de items de menú para una app. Típicamente contendrá las acciones de impacto global para la app, como la búsqueda, seteos o preferencias de usuario, login, logout.

Menu XML Layout

Para todos los tipos de menúes soportados, Android provee un tipo de recurso que es el menu, que puede contener los siguientes componetes:

  • <menu> : contenedor de items
  • <item> : un item simple del menú
  • <group> : permite agrupar items

<?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:id="@+id/new_game"
          android:icon="@drawable/ic_new_game"
          android:title="@string/new_game"
          android:showAsAction="ifRoom"/>
    <item android:id="@+id/help"
          android:icon="@drawable/ic_help"
          android:title="@string/help" />
</menu>

Menu item

atributos:

  • android:id : id único para ese item
  • android:icon : referencia al ícono (drawable) a mostrase
  • android:title : título
  • android:showAsAction : especifica la manera es que es mostrado en el action bar (ifRoom, always, etc)

Creando menú de opciones

Para crear un menú de opciones se require redefinir onCreateOptionsMenu(). En dicho método, se puede "inflar" el recurso menú definido en el XML. 


@Override
public boolean onCreateOptionsMenu(Menu menu) {
    MenuInflater inflater = getMenuInflater();
    inflater.inflate(R.menu.game_menu, menu);
    return true;
}

Manejando las acciones

Cuando el usuario selecciona un item del menú, el sistema invoca el método del Activity::onOptionsItemSelected(). 


@Override
public boolean onOptionsItemSelected(MenuItem item) {
    // Handle item selection
    switch (item.getItemId()) {
        case R.id.new_game:
            newGame();
            return true;
        case R.id.help:
            showHelp();
            return true;
        default:
            return super.onOptionsItemSelected(item);
    }
}
